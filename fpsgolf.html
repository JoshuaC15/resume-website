FPS Golf

Unity C# Xcode

A mobile game that combines the best aspects of first person shooters with sports games. Made with Unity, key features include smart enemy AI, which track the player within a certain range, as well as responsive joystick controls that allow for movement and shooting. It includes 3 fully developed levels, each with unique elements and level designs, as well as a score tracker.
