Slated Social Media Platform

Python Flask HTML CSS Javascript Firebase React SQL

Slated is a social media web application that focuses on “social scheduling”. The main draw is users having a weekly schedule that they update, which outlines what they have to do, as well as what free time they have. Their friends can then see view their schedule and send invites to make plans. This functionality was done with Flask, with an SQL database, while the front end was done with Firebase and React. In addition to this, users are able to make their own account and add friends.
